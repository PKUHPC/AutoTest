Do broadcast and save result into dims_out.

Compares every pair of elements of dims_in1 and dims_in2 in reverse order. Suppose that k and m are the i_th last elements of dims_in1 and dims_in2. (1) k == m, then dim_out[i_th last] = k; (2) k == 1 or m ==1, then dim_out[i_th last] = max(k, m); (3) k != m and k != 1 and m !=1, mismatched error. For example: dims_in1 = {3, 2, 3, 1, 5}, dims_in2 = {1, 3, 4, 5}, then dims_out = {3, 2, 3, 4, 5}

Note
This function does not allocated any memory inside, so the memory of dims_out is assumed to be managed by user.

Applies a batch normalization over an input along axis which is actually the axis of channel.

The data type of input, scale, bias, mean and variance should be identical. The shape of output is same as the shape of input.

Parameters
inputThe input tensor, whose shape should be [N,L], [N,C,L], [N,L,C], [N,C,H,W], [N,H,W,C], [N,C,D,H,W] or [N,D,H,W,C].
axisThe axis along which to be normalized, actually the axis of channel. For instance, if the shape of given input is [N,C,H,W], then axis should be set to 1. But if the shape of input is [N,L], axis should be 1.
scaleGamma in equations. If the shape of input is [N,L], then the shape of scale is [L]; else the shape of scale is [C].
biasBeta in equations. If the shape of input is [N,L], then the shape of bias is [L]; else the shape of bias is [C].
meanThe mean tensor. If the shape of input is [N,L], then the shape of mean is [L]; else the shape of mean is [C].
varianceThe variance tensor. If the shape of input is [N,L], then the shape of variance is [L]; else the shape of variance is [C].
epsilonSmall number added to variance to avoid dividing by zero.
outputThe output tensor pointer.
Returns
Status The Status enum indicates whether the routine is OK.

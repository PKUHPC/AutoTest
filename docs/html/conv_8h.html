Applies a convolution over an input image with a filter.

Parameters
inputThe input tensor of a shape [num_batches, in_channels, in_spatial_shapes].
filterThe filter tensor of a shape [out_channnels, in_channels, in_spatial_shapes].
strideThe stride of a shape [spatial_shapes] each of which is >= 1.
paddingThe padding of a shape [spatial_shapes] each of which is >= 0. The left and right of each dimension use the same padding * width.
dilationThe dilation of a shape [spatial_shapes] each of which is >= 1. This is a regular convolution if all values are 1.
groupsAn integer used to split input into groups, in_channels should be divisible by the number of groups.
outputThe output tensor pointer of a shape [num_batches, out_channels, * * out_spatial_shapes].
Returns
Status The Status enum indicates whether the routine is OK.

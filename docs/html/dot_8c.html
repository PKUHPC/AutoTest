Status aitisa_dot ( const Tensor  tensor1,
const Tensor  tensor2,
Tensor *  output 
)

Applies a dot over two input signals.

If the two inputs are one-dimension, then implement the normal dot; If the two inputs are two-dimension, then implement the matmul; If one of the two inputs is a scalar, then implement the scalar-tensor multiplication; If the second input is one-dimension, then calculate the dot between the last dimension of the first input and the second input; If both of the dimensions of the two inputs are larger than 2, then calculate the dot between the last dimension of the first input and the second-to-last dimension of the second input: dot(a, b)[i,j,k,m] = sum(a[i,j,:] * b[k,:,m]).

Parameters
tensor1The first input tensor.
tensor2The second input tensor.
outputThe output tensor pointer.
Returns
Status The Status enum indicates whether the routine is OK.
